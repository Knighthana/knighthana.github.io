# Visual Studio Code 的多行同时编辑

Visual Studio Code 提供了垂直方向同时使用多个光标以同时编辑多行的功能

按住`Alt`键，鼠标点击相应的行，以在该行上进行增加或删除一个光标的操作

*虽然不知道有什么用，但是觉得这很酷*
